# Moses Nyanzi | Strategy. Logic. Results.

Personal consulting website for Moses Nyanzi - Actuary | MBA. Strategic advisory for senior leaders in insurance, private equity, and financial services.

## Live Site

[mosesnyanzi.co.za](https://mosesnyanzi.co.za) *(or your deployed URL)*

## Tech Stack

- **Framework:** React 18 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Colour Palette:** Navy, Gold, Forest, Terracotta

## Features

- **Visual-first design** with animated SVG diagrams
- **Smooth scroll progress bar** using Framer Motion springs
- **Responsive** - mobile-optimized layouts
- **Section animations** with staggered entrances
- **Interactive technical diagrams** (Reinsurance Tower, SAM Waterfall)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Type check
npx tsc --noEmit
```
